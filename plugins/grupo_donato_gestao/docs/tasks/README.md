# Tarefas operacionais

Crie um arquivo por tarefa autorizada, com nome estável e ordenável, por exemplo:

```text
YYYY-MM-DD-fase-ou-assunto.md
```

Cada tarefa deve registrar objetivo, escopo, fora de escopo, pré-condições, critérios específicos, evidências esperadas e estado final. Não copie `CONTEXT.md` ou `GUARDRAILS.md`; referencie-os.
